Cara redirect cloudflare ke https

Mas Wahhh July 13, 2019 Plesk

Udah pake cloudflare tapi kok masih belum https? nda aman?

ini biar kalian pengguna cloudflare dapat ssl, dan website awto aman / langsung ngarah ke Https, biar nda berisik.

  • Login ke cloudflare, nda punya? buat dulu Silahkan login dulu ke link sini cloudflare

  • Buat Page rules

  • klik create page rules kaya gambar dibawah,
  • Masih buat page rules

penjelasan

Isi url sama http://*namadomain/* domain mana aja yang didirect mau didirect ke https, pakai wildcard atau tanda * buat identifikasi keseluruhan link ngarah ke HTTPS

Tambah Always Use HTTPS Biar ketika website diakses langsung ngarah / di redirect ke https

  • Setelah dibuat

Silahkan akses, kalau masih nda bisa clear cache browser dulu atau coba akses pakai browser lain
