Install Plesk panel di Centos 7 dalam "Sekali enter"

Mas Wahhh July 13, 2019 Plesk

Plesk is a commercial web hosting platform with a control panel that allows a server administrator to set up new websites, reseller accounts, e-mail accounts and DNS entries through a web-based interface.
“Wikipedia”

yak, Kurang lebih seperti itu penjelasannya. Terima kasih wikipedia atas penjelasannya.

Masih nda paham? Simplenya gini, plesk adalah perangkat lunak yang digunakan untuk melakukan pengaturan pada web hosting, seperti melakukan pengaturan DNS, upload file-file website yang nantinya akan ditampilkan, dan masih banyak lagi.

yaa, kurang lebih seperti itu. Kalau masih kurang jelas jangan panik.

Google diciptakan untuk mempermudah pekerjaan kalian.

Udah banyak banget penyedia hosting dari indonesia yang menawarkan paket hosting menggunakan Plesk. Banyak! tinggal pilih sesuai kekuatan masing-masing.

Tapi kalau punya vps dan atau sekedar mau coba-coba aja bisa kok. Bisa dapat trial 15 Hari coba Plesk Onyx yang langsung disediakan sama plesk nya. Kalau cara installnya udah dijelasin lengkap di link ini

https://docs.plesk.com/en-US/onyx/deployment-guide/plesk-installation-and-upgrade-on-single-server/1click-plesk-installation/installing-plesk-for-linux-in-one-click.76444/

Cari yang paling simple buat install nya, paste, tunggu selesai. Iya diatas itu dokumen atau panduan yang disediain sama plesk nya buat install plesk 1 click. Yaa kalau nda terima 1 click bisa sebut 1 enter lah.

Beneran 1 enter kok, coba login dulu ke VPS. Pastiin akses root ya, terus paste perintah,

# sh <(curl https://autoinstall.plesk.com/one-click-installer || wget -O - https://autoinstall.plesk.com/one-click-installer)

bis itu “Enter” dengan penuh kasih sayang please.

Tungguin sampai selesai deh.
